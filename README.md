# Platra

## Install 

```
$ docker-compose up
$ docker-compose run app rake db:create
$ docker-compose run app rake db:migrate
```

## Run Platra

```
http://localhost:3000
```

## Run Tests

```
$ docker-compose run app rails test
```

## Run React Frontend

```
$ cd platra_web_reactjs_rewrite
$ npm start
http://localhost:3000
```

## Run phpMyAdmin
